Загальні положення

Це специфікація визначає можливості та синтаксис для Scalable Vector Graphics (SVG) Версії 1.1, модульної мови для опису двовимірної векторної та змішаної, векторно/растрової графіки XML.

Статус цього документу

Цей розділ описує статус цього документу на час публікації. Інші документи можуть замінювати цей документ. Список поточних публікацій W3C та останні ревізії цього технічного повідомлення можуть бути знайденими в W3C technical reports index за адресою http://www.w3.org/TR/.

Цей документ від 16 Серпня 2011 є SVG 1.1 Рекомендаціями другої редакції. Друга редакція вклчає в себе деяку кількість виправлень, що були опубліковані як errata відносно першої редакції, а також численні зміни, що допомагають зробити специфікацію більше легкою для читання та недвозначною. Додаток Зміни перелічує всі зміни, що були зроблені з часу першої публікації Пропонованих Рекомендацій в другій редакції. Для пошуку всіх змін між Першою редакцією та Другою редакцією, дивіться:

  • Зміни між SVG 1.1 Перша редакція Рекомендацій та першою Робочою чернеткою останнього дзвінка SVG 1.1 Друга редакція

  • Зміни між першою та другою Робочою чернеткою останнього дзвінка SVG 1.1 Друга редакція

  • Зміни між другою Робочою чернеткою останнього дзвінка та Пропонованими рекомендаціями SVG 1.1 Друга редакція

  • Зміни між Пропонованими рекомендаціями ті цим документом

Коментарі щодо цих Рекомендацій бажані. Виправлення відносно специфікації буде опубліковано як errata, та в подальшому буде об’єднано в наступні редакції SVG 1.1 або в SVG 2.0. Коментарі можуть бути надіслані на www-svg@w3.org, публічний список розсилки для проблем, пов’язаних з векторною графікою в веб. Цей список архівовано та відсилачі мають погодитись, що їх повідомлення публічно архівовано починаючи з їх першого надсилання. Щоб підписатися, надішліть лист електронної пошти на www-svg-request@w3.org зі словом subscribe в рядку теми.

W3C SVG Working Group випустила розширений тестовий комплекс для SVG 1.1 разом з повідомленням щодо реалізації. Цей тестовий комплекс буде продовжувати оновлюватися за рахунок нових тестів для покращення взаємодії, навіть після фази Рекомендацій.

Цей документ був продукований W3C SVG Working Group як частина Графічної активності в рамках W3C Interaction Domain. Цілі W3C SVG Working Group обговорюються в W3C SVG Charter. W3C SVG Working Group підтримує публічну веб сторінку , http://www.w3.org/Graphics/SVG/, що містить подальшу фонову інформацію. Автори цього документу є учасниками SVG Working Group.

Цей документ було переглянуто членами W3C, розробниками програмного забезпечення, та іншими групами W3C та зацікавленими сторонами, та затверджено директором як Рекомендації W3C. Це стабільний документ та може бути використаний як матеріал для посилання або цитований з другого документу. Роль W3C's в виробленні Рекомендацій є привертання уваги до специфікації та просування в більш поширену розробку. Це розширює функціональність та сумісність в веб.

Цей документ було вироблено групою, що діє під впливом Патентної Політики W3C від 5 Лютого 2004. W3C підтримує публічний список будь-яких патентних розголошень, зроблених у зв’язку з витоками групи; ця сторінка також включає інструкції для розголошення патенту. Індивідуум, що має фактичне знання патенту, що, за його переконанням, містить Суттєві вимоги, має розкрити інформацію відповідно до розділу 6 Патентної Політики W3C.
